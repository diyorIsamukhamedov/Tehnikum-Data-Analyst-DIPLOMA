В созданном отчете с подключенными источниками создать

1. Вычисляемый столбец (флаг опоздания) в таблице flight по следующей логике

    Если actual_departure <= shedule_departure - Рейс вылетел вовремя
    Если actual_departure > shedule_departure - Рейс вылетел с опозданием
    Если  actual_departure пустое - Рейс еще не выполнен

    Подсказка, для проверки на пустоту нужно воспользоваться функцией isblank или проверить на равенство с blank(), для возвращения определенного значения по условию необходимо воспользоваться функцией if(). Также, обратите внимание, что функции if можно вкладывать друг в друга

2. Меру, которая считает % рейсов, которые вылетел вовремя к общему количеству рейсов
